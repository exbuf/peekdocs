Notes
═══════════════════════════════════════════════════════════════════

- The (415) 555-0123 phone uses the 555 reserved-fiction prefix —
  not a real number.
- Bates Number "SMITHCO000582" requires no separator between letters
  and digits, which is why other identifiers like PO-22781 don't
  accidentally trigger it.
- "MLS#8821055" is written without a space because the regex allows
  exactly one separator character (# or whitespace), not both.
- Invoice "INV-48921" is single-segment for the same reason — the
  pattern stops at the first non-digit, so "INV-2026-0042" would
  only highlight up through "INV-2026".
